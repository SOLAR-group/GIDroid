Use esta opção para manter a tela ligada em um determinado nível de brilho quando coletando medições. A tela permanecerá ligada mesmo se você trocar para outra aplicação ou tentar desligá-la usando o botão de Power. Habilitá-la irá reduzir significativamente a duração da bateria.

Esta pode ser uma forma de contornar o problema de manter detalhes das células e níveis de sinal atualizados em alguns dispositivos quando usando a API 1.0 do Android.

Para manter a tela ligada somente quando a tela principal está visível, veja a opção na seção Exibição.
