O propósito

O aplicativo foi projetado para coletar coordenadas GPS correlacionadas com informações sobre as torres celulares. Ele foi projetado para ajudar a mapear a extensão da cobertura da rede celular.

Como começar

Para iniciar selecione a opção 'Iniciar coleta' (botão gravar) a partir do menu no topo da tela. As medições serão coletadas somente quando você está na área de alcance das redes móveis e o GPS te dá um posicionamento com pelo menos 40 metros de precisão.

Próximos passos

Os dados coletados podem ser enviados para os bancos de dados do OpenCellID.org e do Mozilla Location Services ou exportados para um arquivo. Antes do primeiro upload você precisa selecionar quais bancos de dados você deseja contribuir. No caso do OpenCellID.org você tambpem precisará registrar gratuitamente e entrar com a chave de API recebida nas Preferências. As medições serão apagadas do teu dispositivo após o envio com sucesso.
