Ключ API (токен) - це спеціальна рядок, що використовується для авторизації користувача для завантаження зібраних вимірювань у базу даних OpenCellID.

Правильний ключ повинен складатися з цифр 0-9 та літер a-f. Залежно від дати реєстрації формат ключа може виглядати наступним чином:

  • {}9743a66f914cc249efca164485a19c5c{} – 32 символи;
  • {}9743a66f-914c-c249-efca-164485a19c5c{} – 32 символи в п'яти групах, розділених дефісом, всього 36 символів.
  • 9743a66f914cc2 & # 8211; 14 символів;
