O OpenCellID é um projeto de código aberto, com o objetivo de criar um banco de dados mundial de CellIDs correlacionadas com posicionamentos GPS. Os dados coletados estão disponíveis sob a licença "Creative Common Attribution-Share Alike 3.0 Unported".
O Tower Collector te dá a oportunidade de contribuir com o projeto OpenCellID ao enviar das torres celulares da tua área.Mais informações.
