Mozilla Location Services (MLS) è un progetto open source, con l'obiettivo di creare un database mondiale di identificatori di infrastruttura di rete wireless (torri cellulari, punti di accesso WiFi, beacon Bluetooth) correlati a posizioni GPS. L'insieme aggregato delle celle è disponibile con la licenza "Creative Commons (CC-0)" di pubblico dominio.
Tower Collector ti dà l'opportunità di contribuire al progetto Mozilla Location Services caricando le torri cellulari dalla tua zona. Ulteriori informazioni.
