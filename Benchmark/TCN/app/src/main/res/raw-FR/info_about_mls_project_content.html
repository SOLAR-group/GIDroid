Mozilla Location Services (MLS) est un projet open source, qui a pour but de créer une base de données mondiale d'identificateurs d'infrastructures de réseaux sans fil (antennes relais, points d'accès WiFi, balises Bluetooth) en relation avec les emplacements GPS. L'ensemble des données de localisation des cellules est disponible dans le domaine public sous la licence "Creative Commons (CC-0)".
Tower Collector vous donne la possibilité de contribuer au projet Mozilla Location Services en téléchargeant les données des antennes relais de votre région. Plus d'informations.
