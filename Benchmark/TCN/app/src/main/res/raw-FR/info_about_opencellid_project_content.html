OpenCellID est un projet open source, qui a pour but de créer une base de données mondiale de CellID en relation avec les positions GPS. Les données collectées sont disponibles sous la licence "Creative Common Attribution-Share Alike 3.0 Unported".
Tower Collector vous offre la possibilité de contribuer au projet OpenCellID en chargeant les antennes relais de votre région. Plus d'informations.
