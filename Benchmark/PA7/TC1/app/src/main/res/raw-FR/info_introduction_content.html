L'objectif

L'application est conçue pour collecter des coordonnées GPS corrélées avec des informations sur les tours cellulaires mobiles. Il est conçu pour aider à cartographier l'étendue de la couverture du réseau téléphonique mobile.

Comment démarrer ?

Pour commencer, sélectionnez L'option "Démarrer la collecte" (icône d'enregistrement) dans le menu en haut de l'écran. Les mesures ne seront collectées que lorsque vous êtes à portée de réseau mobile et que le GPS donne votre position avec une précision d'au moins 40 mètres.

Prochaines étapes

Les données collectées peuvent être envoyées vers les bases de données OpenCellID.org et Mozilla Location Services ou exportées dans un fichier. Avant le premier envoi, vous devrez sélectionner les bases de données auxquelles vous souhaitez contribuer. Dans le cas de OpenCellID.org, vous devrez en plus vous inscrire gratuitement et entrer la clé API reçue dans les préférences. Les mesures seront supprimées de votre appareil après un envoi réussi.
