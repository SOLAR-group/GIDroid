Mozilla Location Services (MLS) это проект с открытым исходным кодом, придуманный для создания всемирной базы данных идентификаторов беспроводных сетей (сотовых вышек, WiFi точек доступа, Bluetooth маяков) с привязанными к ним GPS координатами. Собранные наборы данных доступны под лицензией "Creative Commons (CC-0)".
Tower Collector позволяет вам поддержать проект Mozilla Location Services путём выгрузки сотовых вышек в вашем регионе. Больше информации.
