OpenCellID es un proyecto de código abierto, con el objetivo de crear una base de datos mundial de CellID correlacionada con ubicaciones GPS. Los datos recopilados están disponibles en la licencia "Creative Common Atribution-share igual 3,0 Unported ".
Tower Collector le da la oportunidad de contribuir al proyecto OpenCellID cargando torres de celdas de su área. Más información.
