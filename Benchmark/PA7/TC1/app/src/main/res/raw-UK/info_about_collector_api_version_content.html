Проте деякі виробники не застосовували правильно новий API, і вони можуть не використовуватися на таких пристроях. У версії 5.1 та новіших стільникових телефонів інформація може бути отримана з декількох SIM-карт.
  • Android 1.0 API - використовує набір методів, доступних після першої версії Android. Ці методи можуть повернути ненадійні дані для мереж LTE та UMTS. Залежно від пристрою цей метод може вимагати, щоб екран залишався увімкненим під час збору, щоб зберегти правильну інформацію про соту.
