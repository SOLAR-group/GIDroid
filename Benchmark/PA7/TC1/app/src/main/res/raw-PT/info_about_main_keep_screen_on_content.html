Use esta opção para manter a tela ligada no brilho máximo quando a tela principal é apresentada no dispositivo, Assim que você deixa a tela ou troca para uma aplicação diferente o comportamento definido no Android será restaurado. Habilitá-la irá reduzir significativamente a duração da bateria.

Para manter a tela ligada todo o tempo quando coletando medições, veja a opção na seção Coletor.
