A chave de API (token) é uma string especial usada para autorizar o usuário a fazer o upload das medições coletadas no banco de dados do OpenCellID.

A chave correta deve consistir de dígitos de 0-9 e letras de a-f. Dependendo da data do registro o formato da chave pode parecer assim:

  • 9743a66f914cc249efca164485a19c5c – 32 caracteres;
  • 9743a66f-914c-c249-efca-164485a19c5c – 32 caracteres em cinco grupos separados por hífen, total 36 caracteres.
  • 9743a66f914cc2 – 14 caracteres;
