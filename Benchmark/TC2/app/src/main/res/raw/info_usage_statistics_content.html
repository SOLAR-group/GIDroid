Usage statistics is a form of anonymous feedback. It help developer to better understand how application is being used, how fast it is and what are most popular features. With this kind of knowledge it is much easier to improve overall performance, usability and user experience.

Tower Collector tracks and reports the following kinds of data:

  • application failures;
  • which features are used and how;
  • time consuming operations duration;
  • number of collected, exported and uploaded measurements (statistics only).

Real measurements data, API key and any other personal information is NOT being tracked. Reports are send automatically in the background. If you treat your privacy very seriously, you can easily opt-out from collecting and sending usage statistics by unchecking option in Preferences.
